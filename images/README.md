## Image Guidelines

- **Format**: JPG or PNG
- **Quality**: High resolution, HD quality
- **Theme**: Nature-inspired, eco-forest theme with greenish tones
- **Style**: Premium, clean, modern, futuristic
- **Optimization**: Images should be optimized for web (compressed but maintaining quality)
